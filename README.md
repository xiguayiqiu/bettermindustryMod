# Better Mindustry — 更好的像素工厂

> 作者：弈秋忘忧白帽
> 版本：1.0.0
> 最低游戏版本：v158

## 概述

全面提升 Mindustry 游戏体验的综合向模组，集成了多项实用功能，所有功能均可通过 **BM 设置**（游戏设置 → BM设置）独立开启/关闭。

## 功能列表

所有功能均支持在 BM 设置中独立开关，部分功能需重启游戏生效。

### 1. 内置楷体字体

- 在游戏主界面中替换默认字体为内置楷体
- 适合中文显示，效果更美观
- 由 Java 设置 + JS 脚本联合控制，启动时注入字体文件
- 需重启游戏生效

### 2. 显示未解锁科技

- 在科技树中显示未解锁的内容（灰色图标）
- 点击不会解锁，仅作预览
- 需重启游戏生效

### 3. 游戏内时间流速控件

- 在游戏界面左下角添加时间流速控制面板
- 支持：1×、16×、32×、64× 四档速度
- 仅调整游戏运行速度，不影响单位、规则或胜负判定
- 面板可拖动
- 需重启游戏生效

### 4. 游戏内核心资源面板

- 在游戏界面左上角显示核心资源面板
- 实时显示当前核心所拥有的资源及数量
- 显示核心数量、在线玩家、采矿单位数量
- 面板可拖动
- 需重启游戏生效

### 5. 自定义加载界面 \[功能实验性测试]

- 替换原版加载界面为自定义样式
- 可选显示阶段文字、当前加载的 Mod 列表、内存用量
- 进度条动画速度可调
- 可在设置中完全关闭以使用原版加载界面

## 设置说明

所有设置位于 **游戏设置 → BM设置**（带扳手图标），目前包含以下开关：

| 设置项       | 设置键                    | 默认值 | 需重启 |
| --------- | ---------------------- | --- | --- |
| 使用内置楷体字体  | `bm-kai-font-enabled`  | 关闭  | 是   |
| 显示未解锁科技   | `bm-showlocked`        | 关闭  | 是   |
| 游戏内时间流速控件 | `bm-timeflow-enabled`  | 关闭  | 是   |
| 游戏内核心资源面板 | `bm-coreitems-enabled` | 关闭  | 是   |

> 注：由于 JS 脚本在游戏启动时一次性执行，所有依赖 JS 的功能均需重启游戏。

## 构建

本模组同时支持桌面版和 Android 版构建。

```bash
# 构建桌面版
./gradlew jarDesktop

# 构建 Android 版（需要 Android SDK）
./gradlew jarAndroid

# 或使用构建脚本
./build_android.sh
```

构建产物位于 `build/libs/` 目录下。
